# 🎓 eStudy - Education Landing Page

eStudy is a modern and responsive education landing page built using React.js and Tailwind CSS. It is designed to present educational content in a clean, user-friendly interface with smooth navigation and fully responsive layouts across all devices.

---

## 🚀 Features

- 📱 Fully responsive design (mobile, tablet, desktop)
- ⚡ Fast and optimized performance
- 🎯 Clean and modern UI
- 🧭 Smooth navigation between sections
- 📚 Well-structured sections:
  - Home
  - About
  - Courses
  - Reviews
  - Contact

---

## 🛠️ Built With

- React.js  
- Tailwind CSS  
- JavaScript (ES6+)  
- HTML5  
- CSS3

## ⚙️ Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/eslamahmedbadea/eStudy_Education_Landing_Page_React.git
```

2. Navigate to the project folder:
```bash
cd eStudy_Education_Landing_Page_React
```

3. Install dependencies:
```bash
npm install
```

4. Run the development server:
```bash
npm run dev
```

## 🎯 Purpose

This project was built for learning and practicing frontend development using React and Tailwind CSS, focusing on creating responsive and visually appealing landing pages.
